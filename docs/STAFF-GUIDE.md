## Getting Started as Staff

### Staff Account Access

1. **Staff Designation**: Your account must be marked as a staff account
2. **Enhanced Permissions**: Staff accounts have access to:
   - All client accounts and data
   - Template creation and editing
   - Administrative functions
   - AI analysis features

### First Login

1. **Email-Based Authentication**: Enter your staff email address
2. **OTP Verification**: Receive and enter the 6-digit verification code
3. **Automatic Dashboard Redirect**: Staff users are automatically directed to the staff dashboard

## Staff Dashboard Overview

### Staff Interface Differences

Unlike client users, staff members see:
- **No Account Selector**: Access to all clients without switching
- **Extended Sidebar**: Additional menu options for Clients and Templates
- **Recent Activity Dashboard**: Overview of recent system activity instead of client-specific metrics

### Sidebar Navigation

Staff users have access to:
- **Dashboard**: Staff activity overview and recent actions
- **Work Items**: Manage work items across all clients
- **Documents**: View documents from all client accounts
- **Clients**: Client account management interface
- **Templates**: Create and manage questionnaire templates

### Staff Dashboard Features

#### Recent Activity View
- **Task Completions**: Track when clients complete tasks
- **Document Uploads**: Monitor new document submissions
- **Message Activity**: See recent chat interactions
- **Timeline View**: 30-day activity history with timestamps
- **Activity Icons**: Visual indicators for different activity types

#### Activity Types
- **Task Completed**: Clients finishing assigned tasks
- **Document Uploaded**: New documents added to client accounts
- **Message Sent**: Chat communications between staff and clients

## Managing Clients and Accounts

### Accessing Client Management

1. **Sidebar**: Click "Clients" to access client management
2. **Client Overview**: View all client accounts and their status
3. **Account Types**: See both Personal and Business accounts for each client

### Client Account Information

For each client account, staff can view:
- **Account Name**: Client business or personal account name
- **Account Type**: Personal or Business classification
- **Contact Information**: Associated user details
- **Work Item Status**: Current tasks and their completion status
- **Document Library**: All uploaded documents and files

### Client Communication

1. **Direct Messaging**: Initiate chat conversations with clients
2. **Task Assignment**: Create and assign new work items
3. **Document Requests**: Request specific documents from clients
4. **Status Updates**: Communicate progress and updates

## Work Item Management

### Viewing All Work Items

1. **Work Items Page**: Access comprehensive view of all client work items
2. **Staff Filtering**: Filter work items by:
   - Client account
   - Task status (pending, completed, urgent)
   - Task type (document, questionnaire, chat)
   - Search by work item or task name

### Work Item Assignment

Staff can assign work items to:
- **Specific Clients**: Target individual client accounts
- **Team Members**: Assign to other staff members
- **Multiple Tasks**: Create complex work items with multiple task types

### Task Types Available to Staff

1. **Document Tasks**: Request specific documents from clients
2. **Questionnaire Tasks**: Assign template-based forms
3. **Chat Tasks**: Create communication threads for specific topics

### Work Item Status Management

#### Status Levels
- **Pending**: Awaiting client action
- **Urgent**: Requires immediate attention
- **In Progress**: Client actively working on tasks
- **Complete**: All tasks finished
- **Done**: Work item closed and archived

#### Status Updates
- **Manual Changes**: Staff can update status manually
- **Automatic Progression**: Some statuses update automatically based on client actions
- **Client Notifications**: Status changes trigger notifications to clients

### Advanced Work Item Features

#### Multi-Client Projects
- **Cross-Account Tasks**: Assign tasks spanning multiple client accounts
- **Team Collaboration**: Coordinate with other staff members
- **Deadline Management**: Set and track due dates

#### Bulk Operations
- **Mass Assignment**: Assign similar tasks to multiple clients
- **Status Updates**: Update multiple work items simultaneously
- **Template Application**: Apply questionnaire templates to multiple clients

## Creating and Managing Templates

### Template Access

1. **Staff-Only Feature**: Templates are only accessible to staff users
2. **Navigation**: Use the "Templates" option in the sidebar
3. **Template Library**: View all existing templates

### Creating New Templates

1. **Create Button**: Click "Create Template" on the Templates page
2. **Template Information**:
   - **Name**: Descriptive template title
   - **Description**: Optional explanation of the template's purpose
   - **Questions**: Add various question types

### Question Types

#### Available Question Types
1. **Text Input**: Short text responses
2. **Long Text**: Extended text areas
3. **Multiple Choice**: Single selection options
4. **Checkbox**: Multiple selection options
5. **Date Fields**: Date picker inputs
6. **File Upload**: Document attachment fields
7. **Number**: Numeric input fields

#### Question Configuration
- **Required Fields**: Mark questions as mandatory
- **Help Text**: Provide guidance for complex questions
- **Validation Rules**: Set specific input requirements
- **Conditional Logic**: Show/hide questions based on previous answers

### Template Management

#### Editing Templates
1. **Edit Mode**: Modify existing templates
2. **Version Control**: Track template changes over time
3. **Active Status**: Enable/disable templates for client use

#### Organizing Templates
- **Categories**: Group templates by tax year, service type, or client segment
- **Search**: Find templates quickly by name or content
- **Usage Analytics**: Track how often templates are used

### Assigning Templates to Clients

1. **Work Item Creation**: Include questionnaire tasks in work items
2. **Template Selection**: Choose appropriate template for client needs
3. **Customization**: Modify template questions for specific clients
4. **Due Dates**: Set completion deadlines for questionnaires

## AI Document Analysis Features

### AI Analysis Overview

The platform includes AI-powered document analysis to help staff process client documents more efficiently.

### AI Analysis Features

#### Automatic Analysis
- **Document Upload Triggers**: AI analysis runs automatically when clients upload documents
- **Content Recognition**: Extract key information from financial documents
- **Data Categorization**: Automatically categorize expenses and income
- **Error Detection**: Flag potential discrepancies or missing information

#### Analysis Indicators
- **Sparkles Icon**: Indicates AI analysis is available for a document
- **Staff-Only Feature**: AI analysis is only visible to staff users
- **Task Integration**: Analysis appears in document task previews

### Using AI Analysis

1. **Document Tasks**: Click on document tasks to view AI analysis
2. **Analysis Preview**: Review extracted data and insights
3. **Verification**: Confirm or correct AI-identified information
4. **Integration**: Use analysis to populate tax forms or accounting records

### AI Analysis Best Practices

#### Review and Verification
- **Human Oversight**: Always verify AI analysis results
- **Cross-Reference**: Check AI findings against original documents
- **Client Confirmation**: Confirm unusual findings with clients

#### Workflow Integration
- **Efficiency Gains**: Use AI to speed up initial document review
- **Focus Areas**: Direct attention to flagged items or discrepancies
- **Quality Assurance**: Maintain high accuracy standards
